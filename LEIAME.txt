1) Instalar o Angular: npm install -g   @angular/cli@7.0.5
2) Instalar o Material Angular: npm i ngx-material-file-input
3) Criar a tabela no Banco de dados Importacao com: Data Source=.\\SQLEXPRESS com o script que está na pasta SQL 
4) Executar o back-end Webapi_Importação
5) Executar o front-end usando no prompt o seguinte comando na pasta importacao-app: ng serve --proxy-config proxy.conf.json 
6) Fazer a chamada do front-end no browser http://localhost:4200/Importacao
